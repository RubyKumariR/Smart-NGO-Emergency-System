# 🚨 Smart NGO Emergency Response System

A full-stack web application that connects NGOs with volunteers for real-time emergency response and task management.

The platform enables NGOs to post emergency cases and allows volunteers to view nearby incidents, accept tasks, and track their progress. It includes secure authentication using JWT, location-based case filtering, interactive maps using Leaflet.js, and a dynamic dashboard for efficient coordination.

**Tech Stack:** HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, JWT, Leaflet.js
